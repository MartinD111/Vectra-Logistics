# Phase 7 — Engine Foundation + Page Registry

**Goal:** the generic block engine + a populated, **compile-time-exhaustive** page registry exist, wired so `pageBlockRegistry.render` produces byte-identical output — with zero change to any `components/projectPage/` file. (ENG-01/02/03)

Reference: `~/.claude/plans/i-think-this-should-expressive-volcano.md` (Phase 7 section + contract).

## Tasks

### 07-01-01 — Generic engine (`lib/workspaceEngine/`)
- `types.ts`: `PluginSource`, `WorkspaceBlockPlugin<B,Ctx>` (native `renderer`/`editor`/`settings` component slots + optional `manifest`), render/edit/settings prop interfaces. Import `PluginBlockManifest` from `@/lib/miniProgram/plugins/manifest`.
- `registry.tsx`: `WorkspaceBlockRegistry<B,Ctx>` class — `get`, `list`, `keys`, `render(block,ctx)` (native → carried renderer; manifest/unknown-with-renderManifest → `renderManifest`; else `null` = today's `default`), `renderEditor(block,ctx,onUpdate)` (carried `editor` else fall back to `render`).
- `index.ts`: re-export both.
- **read_first:** `apps/workspaces/src/lib/miniProgram/plugins/manifest.ts`.
- **acceptance:** `tsc --noEmit` passes; `render` returns `null` for a plugin with no `renderer` and no `renderManifest`.

### 07-01-02 — Page registry adapter (`lib/projectPage/registry.tsx`)
- `PageCtx = { projectId?; clientId?; onChange?; slashItems?; onSlashSelect? }`.
- Build `entries` as an **explicit `Record<PageBlockKind, WorkspaceBlockPlugin<PageBlock, PageCtx>>` literal** (via a `native(kind)` helper pulling metadata + `create` from `pageBlockDef(kind)!`) — the Record literal is the compile-time exhaustiveness proof (a missing kind fails `tsc`).
- **Non-invasive renderer:** every entry's `renderer` delegates to the existing switch component `PageBlockView` (`{block,ctx} => <PageBlockView block projectId clientId onChange />`). Keeps `PageBlockView.tsx` untouched; non-circular (PageBlockView does not import the registry this phase).
- `pageBlockRegistry = new WorkspaceBlockRegistry<PageBlock,PageCtx>(entries, b => b.kind)`.
- **read_first:** `apps/workspaces/src/lib/projectPage/blocks.ts`, `apps/workspaces/src/components/projectPage/PageBlockView.tsx`, `apps/workspaces/src/lib/projectPage/slashMenu.ts`, `apps/workspaces/src/components/projectPage/EditableRichText.tsx`.
- **acceptance:** `pageBlockRegistry.keys().length === 30`; the `Record<PageBlockKind,…>` literal compiles (all kinds present); no file under `components/projectPage/` changed.
