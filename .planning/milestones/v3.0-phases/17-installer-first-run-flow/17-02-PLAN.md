| T-17-06 | Denial of Service | `probeOllamaEndpoint()` | mitigate | `axios.get(..., { timeout: 3000 })` bounds the outbound call so a hung/unreachable endpoint cannot stall the installer indefinitely |
| T-17-07 | Information Disclosure | outbound probe request | accept | The probed URL is operator-supplied during a trusted local install session, not external/attacker-controlled input -- no SSRF concern distinct from the operator's own intent |
| T-17-08 | Information Disclosure | `company_ai_config.local_endpoint`/`local_model` | accept | Local endpoint URL/model name are non-secret operational config -- `api_key_enc` is explicitly passed `null` for local providers (no cloud key ever stored for this path), matching `005_ai_config.sql`'s design |
| T-17-SC | Tampering | npm/pip/cargo installs | accept | No new package installs in this plan -- `axios` and `aiRepository` are pre-existing, already-installed/in-repo dependencies |
