    `const authRateLimiter = rateLimit({ windowMs: 15 * 60 * 1000, limit: 20, standardHeaders: 'draft-8', legacyHeaders: false, handler: (req, res) => { res.status(429).json({ error: 'Too many requests. Please try again later.' }); } });`

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-20-04 | Spoofing (credential stuffing) | `POST /api/auth/login` | mitigate | `authRateLimiter` — 20 req/15min per IP (Task 1) |
| T-20-05 | Denial of Service (signup spam / resource exhaustion) | `POST /api/auth/signup` | mitigate | Same `authRateLimiter` applied to `/signup` (Task 1) |
| T-20-06 | Denial of Service (forgot-password email bombing a victim's inbox) | `POST /api/auth/forgot-password` | mitigate | Same `authRateLimiter` applied to `/forgot-password` (Task 1) |
