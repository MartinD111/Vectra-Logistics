    `function getDaysInMonth(monthStart: Date): number` — returns the number of days in the month `monthStart` falls in (e.g. `new Date(monthStart.getFullYear(), monthStart.getMonth() + 1, 0).getDate()`).

    `function computeBarPosition(startIso: string, endIso: string, monthStart: Date, daysInMonth: number): { leftPct: number; widthPct: number }` — using millisecond-per-day arithmetic (`dayMs = 24 * 60 * 60 * 1000`), compute `clampedStart = Math.max(0, (new Date(startIso).getTime() - monthStart.getTime()) / dayMs)`, `clampedEnd = Math.min(daysInMonth, (new Date(endIso).getTime() - monthStart.getTime()) / dayMs + 1)`, then `leftPct = (clampedStart / daysInMonth) * 100` and `widthPct = ((clampedEnd - clampedStart) / daysInMonth) * 100`. Return `{ leftPct, widthPct }`. This clamps bars that start before or end after the visible month to the grid's edges rather than overflowing.

    Add a minimal placeholder export so the file is a valid, importable module ahead of Task 2: `export function CollectionTimelineView() { return null; }` (Task 2 replaces this stub with the real implementation, keeping `getDaysInMonth`/`computeBarPosition` unchanged).
